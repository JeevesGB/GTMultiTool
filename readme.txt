GT Multi Tool is a gui that features some useful tools and acts as a wrapper for the GT2ModelTool, GT2TextureTool and GT2BillboardEditor exe's.
You WILL NEED these exe's for the tool to run properly. 

How to run: 
Download the repo and place the GTMultiTool folder on the C: drive. 
Open cmd and cd to the folder.
type python main.py
